<h4 align="center">一款用于 qBittorrent 的 ios 客户端</h4>

## 安装步骤
1. 在 Releases 取得 ipa 文件。
2. 使用 [AltStore](https://altstore.io/) 这类软件将 ipa 文件安装到你的设备。

## 系统要求
请确保你的设备系统为 iOS 15.0 以上。
